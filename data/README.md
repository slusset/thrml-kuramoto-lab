# Data catalog

The files in this directory are the published evidence artifacts for the
current lab snapshot. They are generated by scripts in the repository; they are
not external source datasets.

| artifact | generated by | experiment track |
|---|---|---|
| `master_curve.csv` | `run_master.py` | A3 metastable reversal |
| `master_curve.png` | `run_master.py --plot` | A3 metastable reversal |
| `ensemble_placement.csv` | `run_ensemble.py` | A1 ordering and placement |
| `ensemble_hopfield.csv` | `run_ensemble.py` | B1 single-pattern recall |
| `ensemble_capacity.csv` | `run_capacity.py` | B2 multi-pattern interference |
| `ensemble_learned.csv` | `run_learned.py` | B3 learned couplings |
| `ensemble_clock.csv` | `run_clock.py` | C1 equilibrium clock granularity |
| `ensemble_kinetic.csv` | `run_kinetic.py` | C2 kinetic clock granularity |
| `ensemble_capacity_q.csv` | `run_capacity_q.py` | B4 richer relational state |
| `ensemble_fss.csv` | `run_fss.py` | B4 finite-size capacity diagnostic |
| `ensemble_density.csv` | `run_density.py` | B4 density/capacity diagnostic |

The experiment definitions are currently encoded as constants in their
generating scripts. Reproducing a published file therefore requires both the
file's generating script and the matching Git commit. See
[`docs/PROJECT_STRUCTURE.md`](../docs/PROJECT_STRUCTURE.md) for the planned
configuration-manifest format.

New durable experiments should record per-replicate rows before aggregation and
should preserve separate graph, placement, pattern, initial-condition, and
dynamics seeds. Ad hoc plots and temporary run output should go under an
ignored `artifacts/` directory rather than this evidence directory.
